Project Overview

This project aims to predict car prices based on vehicle attributes like make, model, mileage, engine size, and more. Using regression models, I developed a reliable system for estimating car prices accurately.

Methodology

I used the following steps to build the car price prediction model:

  • Data Cleaning: Handled missing values and normalized numerical features.
  • Feature Engineering: Applied one-hot encoding for categorical variables.
  • Modeling: Built and evaluated models using Linear Regression and Decision Tree Regression.

Results

The Linear Regression model achieved the best performance:

  • R-Squared: 0.85
  • Mean Absolute Error (MAE): 2751.16

These results demonstrate the effectiveness of our approach in predicting car prices accurately.
